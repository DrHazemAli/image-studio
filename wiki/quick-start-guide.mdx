## ğŸ¨ Step 3: Create Your First Image

### 3.1 Generate an Image
1. In the studio, locate the **Prompt Box** at the top
2. Enter a descriptive prompt, for example:
   ```
   A beautiful sunset over a mountain landscape with a lake reflection
   ```
3. Select your preferred **AI Model** (DALL-E 3, FLUX 1.1 Pro, etc.)
4. Choose **Image Size** (1024x1024, 1024x1792, etc.)
5. Click **"Generate Image"**

### 3.2 Wait for Generation
- The system will show a loading indicator
- Generation typically takes 10-30 seconds
- Your image will appear on the canvas when ready

## ğŸ”§ Quick Tips

### Essential Shortcuts
- **Ctrl+S**: Save project
- **Ctrl+Z**: Undo
- **Ctrl+Y**: Redo
- **Ctrl+C**: Copy
- **Ctrl+V**: Paste
- **Delete**: Remove selected object
- **Space + Drag**: Pan canvas
- **Ctrl + Mouse Wheel**: Zoom

### Best Practices
1. **Save Frequently**: Use Ctrl+S regularly
2. **Use Layers**: Keep different elements on separate layers
3. **Descriptive Prompts**: Be specific for better AI results
4. **Test Different Models**: Each AI model has unique strengths
5. **Backup Projects**: Export important work regularly
